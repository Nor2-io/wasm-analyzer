Wasmtime is a WebAssembly engine for compiled Wasm modules.

You can find the Wasmtime website [here](https://wasmtime.dev/).

:::note[The ByteCode Alliance]

The Wasmtime runtime project was created by the [Bytecode alliance](https://bytecodealliance.org/). 

:::

<br />

Wasmtime is available for the following languages:

- C
- C++
- Go
- Python
- Rust

<br />

There's also a NuGet package available for the .Net framework, making Wasmtime available for C# and other .Net languages. You can find the NuGet package on the NuGet gallery website, [here](https://www.nuget.org/packages/Wasmtime).
